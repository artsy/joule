# joule

Helpful artsy engineering bot

## Setup

Clone the repo:

```sh
git clone git@github.com:artsy/joule.git
cd joule
```

Install dependencies and setup config values:

```
./scripts/setup.sh
```

## Running

Start bolt server
`yarn start`

Start ngrok (development proxy)
`yarn ngrok`
